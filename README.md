A simple tank game in the terminal based on Tank Trouble.


To install, clone the repo and navigate to it. Run:

`mkdir build`

`cd build`

`cmake ..`

`make`

Start the game:

`./TankTerminal`

Dependencies:
    to run:
        ncurses

    to install:
        cmake 
        compiler for C++20
